# ModelViewer
<br>
archive_dx11 리포지토리에서 분리되었습니다.           
<br>
Dx11로 만든 캐릭터 모델 뷰어입니다.<br>      
최종적으로 게임 내에서 볼 수 있는 캐릭터 화면과 같은 느낌을 내는 것을 목표로 만들고 있습니다.
<br>
<br>

### 참고
사용한 것       
1. fbxsdk(2020ver)      
2. DirectXTex     
<br>
<br>

### 환경
win10, vs19 x86
